Hello, World!

Exploring programming languages, tools, and the foundations of modern software development.

Python

Versatile language known for readability and libraries.

Common uses: Web development, data science, automation

def greet(name):
    return f"Hello, {name}!"

user_name = input("Enter your name: ")
message = greet(user_name)

print(message)

JavaScript

Language of the web, enabling interactive development.

Common uses: Front-end, back-end, mobile apps

function greet() {
  const name = document.getElementById("nameInput").value;
  document.getElementById("output").textContent =
    "Hello, " + name + "!";
}

Java

Robust, object-oriented language used in many enterprise systems.

Common uses: Android apps, enterprise software, big data

class Main {
  public static void main(String[] args) {
    String name = "World";
    System.out.println("Hello, " + name + "!");
  }
}

C++

High-performance language for systems and game development.

Common uses: Games, operating systems, embedded systems

#include <iostream>
#include <string>

std::string greet(std::string name) {
  return "Hello, " + name + "!";
}

int main() {
  std::cout << greet("World");
}

HTML

Standard markup language used to structure web pages.

Common uses: Website structure, content, forms

<!doctype html>
<html>
  <body>
    <h1>Hello, World!</h1>
    <input type="text" placeholder="Enter your name">
    <button>Submit</button>
  </body>
</html>

SQL

Database language used to query and manage structured data.

Common uses: Data retrieval, DB management, backend systems

SELECT first_name, last_name
FROM users
WHERE active = 1
ORDER BY last_name;
